# <a name="skiasharp-bitmaps"></a>SkiaSharp ビットマップ

SkiaSharp の記事「**[ビットマップの基礎」で](../basics/bitmaps.md)** は、ビットマップの読み込みと表示の基本について説明しています。 次の記事では、その入門記事の概念を紹介します。
