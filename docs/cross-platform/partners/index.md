# <a name="certified-consulting-partners-to-deliver-mobile-solutions"></a>モバイルソリューションを提供する認定コンサルティングパートナー

> [!TIP]
> **Xamarin コンサルティングパートナープログラムは、2018年6月30日時点の Microsoft Partner Network に統合されました。**
>
> [Microsoft Partner Network](https://partner.microsoft.com/)の概要:
>
> * まだ登録されている Microsoft Partner Network メンバーでない場合は、パートナーになるように[登録](https://partner.microsoft.com/membership)します。  
> * [アプリケーション開発のコンピテンシー](https://partner.microsoft.com/membership/application-development-competency)や[クラウドプラットフォームのコンピテンシー](https://partner.microsoft.com/membership/cloud-platform-competency)を完成させることによって、専門知識を実証します。
> * [Microsoft ソリューションプロバイダー](https://www.microsoft.com/solution-providers/home)として見つけられるように、Xamarin、Android、iOS、Mobile Apps、モバイルアプリケーションを開発するために、プロファイルの専門タグを更新します。

ビジネスモバイルを活用し、Visual Studio と Xamarin を利用し、より高品質なモバイルソリューションを構築するのに役立つコンサルティングパートナーを見つけることができます。 長期的な戦略的パートナーは、アプリのライフサイクルのあらゆる段階をガイドできるモバイル専門家であり、高パフォーマンスのクロスプラットフォームモバイルアプリを設計、開発、テストするのに役立ちます。

_主な場所別にリストされている多くのパートナーは、境界を越えてサービスを提供しています。_
